# UsTime

为 ISTP & ISFJ 情侣构建的静态单页应用 — 消除决策疲劳，增强情感连接。

## 功能

- 🎲 **随机决策器** — 吃什么、做什么、奖励、情感话题，一键随机决策
- 🏺 **默契储蓄罐** — 存入想说的话，随机抽取惊喜
- 📋 **协作清单** — 游戏化任务列表，带进度条追踪

## 技术栈

Vite 6 + Vanilla JS + Tailwind CSS v4 → GitHub Pages

## 本地开发

```bash
npm install
npm run dev      # 热更新开发服务器
npm run build    # 生产构建
npm run preview  # 预览生产构建
```
