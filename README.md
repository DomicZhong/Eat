# 吃什么 🍽️

按菜系分类的美食随机决策器，帮你决定今天吃什么，消除选择困难。

内置餐厅模式，收录尖沙咀周边真实餐厅，支持一键抽取。

## 功能

- 🎲 **双模式切换**：🍽️菜式 / 📍餐厅，默认抽餐厅
- 🏷️ **6分类筛选**：中式经典🥡 / 港式美食🍜 / 日韩风味🍣 / 东南亚风味🍲 / 西餐异国🍝 / 轻食其他🥗
- 🔄 **换一个**：排除当前结果重新抽取
- 📋 **一键复制**：快速复制结果到剪贴板
- ✏️ **分类管理**：按菜系分组查看、增删改美食/餐厅选项
- 📊 **7天历史**：避免短期内重复推荐
- 🎨 **三主题切换**：🌙 暗夜 / 🌸 暖樱 / 🌿 森林，偏好自动保存
- 💾 **本地存储**：所有数据保存在浏览器 LocalStorage

## 开发

```bash
npm run dev      # 启动开发服务器
npm run build    # 生产构建
npm run preview  # 预览构建产物
```

## 技术栈

- **构建**: Vite 6.x
- **框架**: Vanilla JS (ES6 Modules)
- **样式**: Tailwind CSS v4
- **存储**: LocalStorage
