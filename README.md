# 吃什么 🍽️

按菜系分类的美食随机决策器 — 帮你决定今天吃什么，消除选择困难。

## 功能

- 🎲 **随机抽取**：按分类（全部/中式经典/日韩风味/西餐异国/轻食其他）随机推荐美食
- 🔄 **换一个**：排除当前结果重新抽取
- 📋 **一键复制**：快速复制结果
- 🍽️ **分类管理**：按菜系分组查看、增删改美食选项
- 📊 **7天历史**：避免短期内重复推荐
- 🌙 **暗夜主题**：护眼深色界面

## 开发

```bash
npm run dev      # 启动开发服务器
npm run build    # 生产构建
npm run preview  # 预览构建产物
```

## 技术栈

- **构建**: Vite 6.x
- **框架**: Vanilla JS (ES6 Modules)
- **样式**: Tailwind CSS v4
- **存储**: LocalStorage
